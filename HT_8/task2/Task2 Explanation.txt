Данная задача была решена с помощью Reflection API.
Конструктор класса Tester имеет модификатор доступа private, который подразумевает уровень доступа только внутри самого класса. Для того, чтобы создать объект класса минуя ограничение private, динамически загружаем класс с помощью метода  Class.forName(String), где String - имя класса. В результате получаем объект типа Class. Далее используем метод getConstructor, возращающий объект Constructor, отражающий конструктор класса Tester. Затем мы вызываем метод newInstance() для объекта типа Class, который возвращает объект типа Object. Данный объект приводится к классу Tester и используется для доступа к методам класса Tester.
